No more class components?

Slides available at github.com/vegaasen/presentations (source on GitHub)

Why?

  • Classes, apparently hard to comprehend (this)
  • "Side effect-hell" - unrelated logic for components
  • Splits components in to smaller functions for simplicity (SRP)
  • React docs
  • List of hooks

In essence..

  • Similar to how it works for classes
  • Can have state in functions
  • Can make your own hooks - to be shared between components
  • Only define hooks on top level of functions
  • Only works in functional components
  • Don't use hooks outside of React...
