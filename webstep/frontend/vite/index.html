Treig dev-oppstart

  • Split: Dependencies
    • Pre-bundler med esbuild (førstegangskjøring eller ved update)
  • Split: Kildekode
    • Bruker native ESM
    • Browser funker som en bundler, pga native ESM ✨
    • Vite server kun transformert kildekode, alt er dynamisk! FÆNTASTIC
  • Større/Mellomstore prosjekter - sparer mange sekunder (og frustrasjon)!
